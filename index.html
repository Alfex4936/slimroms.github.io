SlimRoms presents SlimLP Alpha 1.0
Saturday, December 6th 2014 by Abc

We are proud to announce the first weekly for SlimLP. Please note not all devices are getting a weekly yet.This is the first alpha release more devices and features to come as we move forward.

Features:
- RRO Layers Support
- Doze (ambient display) timeout (selected devices only)
- Options for battery icon and text
